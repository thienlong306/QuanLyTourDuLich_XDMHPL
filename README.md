# Tour Manager

Build an MVC Application. Manage tours, groups, customers, employees, costs, locations, group status and statistics.



## Features

- Add
- Remove
- Update
- Search
- Statistics
- Group status

## Installation
```bash
  Import my .sql on xampp
  Open folder TourDuLich
  Add lib mysql-connector.jar
  Run
```
